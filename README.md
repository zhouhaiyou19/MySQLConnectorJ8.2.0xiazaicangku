### 适用场景

- 开发基于 Java 的应用程序，需要连接 MySQL 数据库。

### 使用方法

1. 下载 `mysql-connector-j-8.2.0.zip` 文件。
2. 解压缩文件，获取其中的 `mysql-connector-j-8.2.0.jar`。
3. 将 `mysql-connector-j-8.2.0.jar` 添加到你的 Java 项目的类路径中。
4. 在你的 Java 代码中使用 JDBC 连接 MySQL 数据库。
